== Install ==
Enable module

The new resources are added as relationship to existing resources. For example
if access node object at services/node/1.xml, you'd access to the at 
services/node/1/view.xml. Optionally can pass the params as_page to each and 
it'll render a wrapper head, body, etc. tag that includes the page's javascript 
(use case: embeding everything in an iframe).
